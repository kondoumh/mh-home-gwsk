これは何？

iEdit for Silverlight は、Microsoft の RIA 技術である Silverlight にて実装した、iEdit のクローンです。Windows でも Mac でも動作する iEdit として開発を始めてみましたが、ある程度作ったところで忙しくなり数年放置。そのうちに Microsoft はどうやら Silverlight を見捨てる感じになってきちゃいました。2012年末、Microsoft から Silverlight 終了の悲しいお知らせが。HTML5 に注力するとのこと。ということで、iEdit for Silverlight もその短い開発期間(燃えて作ってたのは数週間)を終えました。

特徴

  • Silverlight で動作する iEdit クローンです。
  • Silverlight 5 をインストールすれば、どこでも使えます。
  • Mac でも使えます。
  • Linux でも Moonlight の開発が進めば使えるようになるかも。
  • Windows 版では実現していないリンクへのリンクも張れます。
  • フルスクリーンモードもあります。
  • グラデーション付きで半透明のノードなどが作れます。
  • アウトオブブラウザーアプリケーションとしてデスクトップアプリケーションのように動作させることが可能です。
  • DirectX 9.0c 以降の DirectX が動作するビデオカードを搭載していれば、GPU Acceleration により描画されます。

ソースコード

kondoumh / iedit-silverlight

制限事項

  • ファイルへの保存、読み込みは不完全です。
  • ファイルへの保存は Mac では動作しません
  • 追加したノードやリンクの削除ができません。
  • リンクは曲げられません。
  • テキストビューはありません
  • ノードやリンクのラベルのインプレース編集はできません(左下のプロパティ設定ウィンドウで編集します)。
  • Macではラベルの編集ができません(なぜ？？)
  • アウトラインビューでのドラッグ＆ドロップはできません(アウトライン操作はツールボタンで)。
  • アウトラインビューでの選択とネットワークビューでのノードの表示・非表示は連動しません。
  • ネットワークビューで表示倍率を小さくすると、編集機能が効かなくなる場合があります
  • ノードやリンクの色や線種などの属性の編集機能はありません。
  • 右クリックのコンテキストメニューは使えません(Silverlight で標準的にサポートされておらず Silverlight の設定画面呼び出しメニューになる)。
  • その他 Windows 版にある様々な機能は未実装です。
